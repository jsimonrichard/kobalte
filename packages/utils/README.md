# @trellis-app/kobalte-utils

Common utilities and types for Kobalte.

## Installation

```bash
npm install @trellis-app/kobalte-utils
# or
yarn add @trellis-app/kobalte-utils
# or
pnpm add @trellis-app/kobalte-utils
```
